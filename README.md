# GigaCorp Support Agent

GigaCorp AI Dashboard is an enterprise-grade customer support interface powered by a Hybrid Retrieval-Augmented Generation (RAG) engine. This system is designed to provide instantaneous, accurate, and context-aware responses to support queries by leveraging specific company knowledge bases.

## Setup
1. Clone this repository.
2. Create a virtual environment: `python -m venv my_ai_env`
3. Activate it and install dependencies: `pip install -r requirements.txt`
4. Run the app: `uvicorn app:app --reload`
